Pruebas de programación Cuántica. Qiskit Language. IBM.

Este repositorio contiene un Jupyter Notebook con programación cuántica, aplicando a un circuito de tres qubits y tres bits clásicos, diferentes tipos de
puertas cuánticas en formato Single-Q y Mult-Q, para observar y cuantificar de forma estadística los diferentes estados de dichos qbits y printearlos en
distintos tipos de grafos utilizando la librería Matplotlib.
La idea principal de este repositorio es la documentación de dicha algorítmia, en forma de iniciación a la libreria de IBM Qiskit.
